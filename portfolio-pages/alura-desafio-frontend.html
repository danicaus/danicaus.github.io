Sobre o projeto

Evento gratuito da Alura, com o desafio de montar uma página a partir de um layout. O evento simulava um ambiente real de trabalho, onde os desenvolvedores deveriam usar suas habilidades para atender o que foi pedido. O objetivo da aplicação era aplicar o highlight em um trecho de código, aplicar uma borda colorida de acordo com o configurado pelo usuário, armazenar essas informações e mostrá-las para o usuário em uma outra página. O armazenamento escolhido foi o LocalStorage, foi necessário usar a biblioteca do Highlight JS.

Tecnologias utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Uso de biblioteca Highlight JS
  • Uso de LocalStorage
  • Interpretação de layout Figma
